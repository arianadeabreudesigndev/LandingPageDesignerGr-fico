short_description:

full_description:
Landing page desenvolvida para uma designer gráfica, apresentando seu trabalho de forma elegante e funcional. O site conta com seções de herói com ilustração personalizada, agenda dinâmica com estatísticas de vagas e projetos finalizados (círculos de progresso), promoção sazonal de landing pages (26% off), cards de serviços com ícones autorais e modal com detalhes, formulário de contato com envio para e-mail via Resend e redirecionamento para WhatsApp, além de rodapé informativo com ícones de redes sociais. Design responsivo, animações suaves com Framer Motion, dark mode e código otimizado com Next.js, TypeScript e Tailwind CSS. Projeto hospedado na Vercel e versionado no GitHub, com foco em conversão e experiência do usuário.
